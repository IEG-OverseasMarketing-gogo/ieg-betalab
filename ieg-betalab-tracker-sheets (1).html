Google Sheets Setup

Follow these steps once. After connecting, all KPI saves, contact adds, and retros write directly to your Sheets.

Step 1 — Create the Google Sheet

1Go to sheets.google.com → create a new spreadsheet named IEG Beta Lab Tracker
2Create four tabs (sheets) named exactly: KPI, Contacts, Retro, PostStatus
3In the KPI tab, add these headers in row 1:
Timestamp | Date | Post | Impressions | Likes | Comments | Followers | PollVotes | LABComments | DMsSent | FormSubmissions | Notes
4In the Contacts tab, add headers in row 1:
Timestamp | Name | Organization | LinkedIn | Stage | DMType | Country | Notes
5In the Retro tab, add headers in row 1:
Timestamp | Date | Week | Content | Timing | Team | KPI | Keep | Improve | Next
6In the PostStatus tab, add headers in row 1:
Post | Status | PublishDate | UpdatedAt

Step 2 — Create the Apps Script

1In the Spreadsheet, go to Extensions → Apps Script
2Delete any existing code. Paste this entire script:
const SS = SpreadsheetApp.getActiveSpreadsheet();

function doPost(e) {
  const data = JSON.parse(e.postData.contents);
  const sheet = SS.getSheetByName(data.sheet);
  if (!sheet) return resp({ok:false, error:'Sheet not found: '+data.sheet});
  sheet.appendRow(data.row);
  return resp({ok:true});
}

function doGet(e) {
  const name = e.parameter.sheet;
  const sheet = SS.getSheetByName(name);
  if (!sheet) return resp({ok:false, error:'Sheet not found: '+name});
  const rows = sheet.getDataRange().getValues();
  return resp({ok:true, rows:rows});
}

function resp(obj) {
  return ContentService
    .createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}
3Click Deploy → New deployment → Type: Web app
4Set "Execute as": Me — "Who has access": Anyone → Deploy
5Authorize when prompted. Copy the Web app URL that appears.
